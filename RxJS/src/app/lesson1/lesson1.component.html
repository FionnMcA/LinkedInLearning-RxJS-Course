Let's say we want to display a snowman for temp below 50 and sun for temp above 50. we can absolutely subscribe to the tenperature changes and react to them. but we dont necessarilly have to do that to display a new weather icon to the user.
Behaviour subjects are a type of observable that allow you to get a current value from a subject.
Qualities of behaviour subjects

  • The initial value is passed to our subscription when we subscribe to them
  • They work like any subject expect you dont have to subscribe to them to keep track of the latest value (but we can if we want to)
  • If all we care about is what the current temperatire is then behaviour subjects are ideal
