For this use case, we'll want to use a regular Subject. This is because we're not interested in the previous or current values.

We'll only start converting the value when the user clicks on the conversion button.

We can't use a BehaviorSubject like before because a BehaviorSubject will emit the current value to every new subscription. Subjects, on the other hand, will not emit anything to new subscribers, only listening to the values that are passed in.
