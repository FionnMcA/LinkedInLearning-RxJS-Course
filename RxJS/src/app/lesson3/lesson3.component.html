    • What if we wanted to convert all previous values as well. we can do this with a special type of observable called a replay subject.

    Replay subjects remember every value thats been passed to it and we'll pass all of them to new subscriptions in the same order.

    Say if we have F values of 21,37,42,69,73 abd we pass them into a replay subject, a current subscription will receive these values
    Newer subscriptions that join later will immedisately be passed the same values

    Behaviour subject: has an inital value and can acess the current value anytime
    Subject: Doesnt have an intial value and passes new data as it recieves it
    Replay Subject: Remembers data passed into it and passes all of that data to new subscribers
